开启我的博客之旅

#建博客心得体会
一分耕耘，一分收获。历经很长时间的踩坑之路之后，我终于成功的搭建了自己的博客，下面，来说说我主要踩的坑吧。希望大家在自己搭建博客的时候可以参考一下。
在下载，安装git时
在这个步骤，主要需要注意的地方有：
1.在注册了自己的github账号之后要登陆自己的邮箱进行验证。
2.在新建代码仓库时，代码仓库的名字要写成yourname.github.io。
在安装Node.js时
在这个步骤，主要需要注意的地方有：
1.在官网下载Node.js的最新版
2.一路next进行安装，在安装之后，要在命令行中输入以下代码进行验证：
node -v
nom -v
在进行与Hexo有关的相关操作时
在这一部分需要注意的地方最多，要避开的坑主要有：
1.尽量要在C盘以外的地盘里创建hexo文件夹。
2.可以先输入以下代码在本地http://localhost进行预览
hexo server
3.在配置-config.yml文件时，要在：之后输入空格。
相信在做好以上几个步骤之后，再参考教程就可以用hexo和github搭建出属于自己的博客了。

cxp的第一篇博客：博客的搭建方法

这种搭建博客的方式主要需要用到git,node.js,以及再把这两个软件安装成功后通过git bash的方式安装。先简单的说一下这种搭建博客的主要步骤。
1.搭建环境准备（主要包括node.js和githuanjing ,github账户的配置）。
2.安装Hexo。
3.配置Hexo。
4.讲hexo 与github page联系起来。
5.在已经完成了博客的搭建之后，就要开始写自己的博客了。
6.为了使博客看起来更美观，以及满足个人的个性化的审美，要为已经搭建好的博客选好合适的主题。
以下是比较详细的论述。
搭建环境准备
大概可以分为3步
.Node.js 的安装和准备。
.git的安装和准备。
.github账户的配置。
配置Node.js环境
在官网中进行下载，一路next安装即可。在安装后通过进入Windows的命令行，分别输入以下两行代码进行验证。
node -v
npm -v
当出现版本号时，代表安装成功。
配置git环境
在官网下载，并且一路next进行安装，也使用和上面一样的方法进行验证。
git账户的注册和配置
这一步比较重要，可以细分为以下几个小步骤。
1.Github注册。进入GitHub的官网http://github.com/进行注册。
2.创建代码库。这一步中，代码库的名称要设置为yourname.github.io,例如，我设置成了sncxpcxp.github.io。
3.代码库设置。
安装Hexo
先在D盘下新建一个Hexo的文件夹，右击进入，选择”git bash”选项，接连输入以下代码以实现安装Hexo并检验Hexo是否安装成功。
npm install hexo-cli -g
npm install hexo --save
hexo -v
hexo的相关配置
接下来再输入以下代码，以继续完成安装Hexo相关组件的工作。
hexo init
hexo install
接下来可以首次体验Hexo并实现预览网页效果。依次输入以下代码：
hexo g
hexo s
这时就可以在浏览器打开http://local4000以实现预览。
将hexo与GitHub page联系起来
主要得经历以下几步：
1.配置git个人信息。
2.配置deployment。
敲黑板啦，以上就已经完成了博客的搭建工作
接下来小伙伴们就可以用markdown的编辑器来写博客，开启自己的博客之旅啦！
