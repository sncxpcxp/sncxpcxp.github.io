开启我的博客之旅

#建博客心得体会
一分耕耘，一分收获。历经很长时间的踩坑之路之后，我终于成功的搭建了自己的博客，下面，来说说我主要踩的坑吧。
在下载，安装git时
在这个步骤，主要需要注意的地方有：
1.在注册了自己的github账号之后要登陆自己的邮箱进行验证。
2.在新建代码仓库时，代码仓库的名字要写成yourname.github.io。
在安装Node.js时
在这个步骤，主要需要注意的地方有：
1.在官网下载Node.js的最新版
2.一路next进行安装，在安装之后，要在命令行中输入以下代码进行验证：
node -v
nom -v
在进行与Hexo有关的相关操作时
在这一部分需要注意的地方最多，要避开的坑主要有：
1.尽量要在C盘以外的地盘里创建hexo文件夹。
2.可以先输入以下代码在本地http://localhost进行预览
hexo server
3.在配置-config.yml文件时，要在：之后输入空格。
相信在做好以上几个步骤之后，再参考教程就可以用hexo和github搭建出属于自己的博客了。
