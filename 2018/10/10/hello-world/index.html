cxp's fisrt blog：博客的搭建方法

这种搭建博客的方式主要需要用到git,node.js,以及再把这两个软件安装成功后通过git bash的方式安装。先简单的说一下这种搭建博客的主要步骤。
1.搭建环境准备（主要包括node.js和githuanjing ,github账户的配置）。
2.安装Hexo。
3.配置Hexo。
4.讲hexo 与github page联系起来。
5.在已经完成了博客的搭建之后，就要开始写自己的博客了。
6.为了使博客看起来更美观，以及满足个人的个性化的审美，要为已经搭建好的博客选好合适的主题。
以下是比较详细的论述。
搭建环境准备
大概可以分为3步
.Node.js 的安装和准备。
.git的安装和准备。
.github账户的配置。
配置Node.js环境
在官网中进行下载，一路next安装即可。在安装后通过进入Windows的命令行，分别输入以下两行代码进行验证。
node -v
npm -v
当出现版本号时，代表安装成功。
配置git环境
在官网下载，并且一路next进行安装，也使用和上面一样的方法进行验证。
git账户的注册和配置
这一步比较重要，可以细分为以下几个小步骤。
1.Github注册。进入GitHub的官网http://github.com/进行注册。
2.创建代码库。这一步中，代码库的名称要设置为yourname.github.io,例如，我设置成了sncxpcxp.github.io。
3.代码库设置。
安装Hexo
先在D盘下新建一个Hexo的文件夹，右击进入，选择”git bash”选项，接连输入以下代码以实现安装Hexo并检验Hexo是否安装成功。
npm install hexo-cli -g
npm install hexo --save
hexo -v
hexo的相关配置
接下来再输入以下代码，以继续完成安装Hexo相关组件的工作。
hexo init
hexo install
接下来可以首次体验Hexo并实现预览网页效果。依次输入以下代码：
hexo g
hexo s
这时就可以在浏览器打开http://local4000以实现预览。
将hexo与GitHub page联系起来
主要得经历以下几步：
1.配置git个人信息。
2.配置deployment。
敲黑板啦，以上就已经完成了博客的搭建工作
接下来小伙伴们就可以用markdown的编辑器来写博客，开启自己的博客之旅啦！
